# AI Support Ticket Analyzer

An AI-powered customer support triage tool built with Python, Claude AI, and Streamlit.

## What it does
- Analyzes customer support tickets using Claude AI
- Classifies tickets by category (billing, technical, account, refund, general)
- Scores urgency from 1-10
- Detects customer sentiment
- Drafts suggested responses
- Flags tickets that need human escalation
- Processes single tickets or bulk CSV uploads
- Generates summary reports

## Technologies Used
- Python
- Anthropic Claude API
- Streamlit
- CSV processing

## How to Run Locally

1. Clone the repo
2. Install dependencies: `pip3 install -r requirements.txt`
3. Set your API key: `export ANTHROPIC_API_KEY='your-key-here'`
4. Run the app: `streamlit run app.py`
